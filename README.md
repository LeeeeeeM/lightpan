# lightpan
轻盘，开源的企业级云盘

## 轻盘特性

+ 数据安全性
存储引擎基于minio，纠删码技术
+ 数据误删除保护机制
底层提供删除保护，防止文件误删除
+ 用户，群组机制
+ 文件夹方式的管理，适合用户习惯
