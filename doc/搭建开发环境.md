# 搭建开发环境

+ 安装golang
+ git clone  https://github.com/light4d/lightpan
+ export GOPATH=系统GOPATH：项目GOPATH（可以看到lightpan项目里，有个GOPATH文件夹，这就是项目GOPATH）
+ cd lightpan/GOPATH/src/github.com/light4d/lightpan
+ go get

## 安装依赖组件

+ mysql

+ minio

## 更改配置文件
